# .NET CI/CD (GitOps)

Pipeline **test → build-image → deploy (Git push)** + ArgoCD sync.

Связанные разделы: [[Запуск проекта]] · [[Схема проекта]]

---

## Cold start

1. `terraform apply` — registry, k3d, ArgoCD, namespace `apps`
2. Репозиторий из `templates/dotnet/` → push в Gitea
3. CI обновляет `k8s/overlays/dev` → ArgoCD выкатывает в k3d

---

## Pipeline

| Job | Действие |
|-----|----------|
| `test` | `dotnet test` |
| `build-image` | `docker build` |
| `deploy` | push образа + commit `newTag` в Git |

---

## Kustomize

```
k8s/
├── base/           # deployment, service
└── overlays/dev/   # CI меняет images[].newTag
```

Namespace `apps` создаёт Terraform, не Git.

---

## Проверка

```powershell
terraform -chdir=terraform output argocd_ui_url
terraform -chdir=terraform output -raw kubeconfig | Out-File -Encoding utf8NoBOM kubeconfig.yaml
kubectl --kubeconfig kubeconfig.yaml get app -n argocd
kubectl --kubeconfig kubeconfig.yaml get pods -n apps
```
