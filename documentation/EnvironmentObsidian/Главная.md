# Локальная Git-платформа

Self-hosted: **Gitea** + **Gitea Actions** + **k3d** + **ArgoCD (GitOps)**. Инфраструктура — Terraform.

## Cold start

1. [[Установка компонентов]] — Docker и Terraform
2. [[Запуск проекта]] — `terraform apply`
3. [[.NET CI-CD]] — репозиторий из `templates/dotnet/` (вручную)

## Структура

```
templates/dotnet/     # .NET + CI (GitOps)
terraform/            # Gitea, runner, registry, k3d, ArgoCD
documentation/        # Obsidian-документация
```

## Провайдеры Terraform

| Провайдер | Назначение |
|-----------|------------|
| `kreuzwerker/docker` | Сеть, тома, образы, контейнеры |
| `SneakyBugs/k3d` | k3d-кластер в Docker |
| `hashicorp/kubernetes` + `helm` | Namespaces, ArgoCD Helm |
| `go-gitea/gitea` | API Gitea (провайдер; admin — bootstrap CLI) |
| `argoproj-labs/argocd` | Repository + Application (GitOps) |

Репозитории и проекты в Gitea создаются **вручную**.
